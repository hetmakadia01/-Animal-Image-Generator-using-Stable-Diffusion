# 🐾 Animal Image Generator using Stable Diffusion

## 🚀 Overview
This project is a Text-to-Image Generator built using a pre-trained diffusion model. It allows users to generate realistic animal images simply by entering a text prompt.

## 🎯 Features
- ✨ Generate animal images from text prompts  
- ✨ Beginner-friendly Google Colab setup  
- ✨ Interactive UI using Gradio  
- ✨ Uses powerful Stable Diffusion model  
- ✨ GPU support for faster image generation  

## 🛠️ Tech Stack
- Python 🐍  
- Diffusers (Hugging Face)  
- Transformers  
- PyTorch  
- Gradio  
- Google Colab  

## 📦 Installation & Setup
```bash
pip install diffusers transformers scipy gradio
```

## ⚙️ How It Works
1. Load model: `runwayml/stable-diffusion-v1-5`  
2. Enter a text prompt  
3. Model generates image  
4. Output shown using Gradio  

## 💡 Usage
- Open notebook in Colab  
- Run all cells  
- Enter prompt  
- Click Generate  

## 🧠 Model Details
- Stable Diffusion v1.5  
- Text-to-image AI model
